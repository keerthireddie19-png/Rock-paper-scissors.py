# Rock Paper Scissors Game

A simple Rock Paper Scissors game developed using Python. In this game, the user plays against the computer. The computer randomly selects Rock, Paper, or Scissors and the program determines the winner based on the game rules.

## Features
- User vs Computer gameplay
- Random choice generation
- Simple and beginner-friendly Python code

## Game Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
